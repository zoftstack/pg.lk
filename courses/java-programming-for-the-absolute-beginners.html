Course Overview

Course Title: JAVA Programming For Absolute Beginners.

Course Description: This course has been built for absolute beginners. it doesn't matter even if the student has no basic idea of programming or at least about the computer basics. The only one requirement is that the student should have a small computer, it might be a desktop or laptop. through the course, you will learn and practice how to write an indutrial level java application step by step from zero to master. A lot of exercises and assignments will be assigned to you to build and get used to programming well.
The course will cover the following sections:

  • Part-1

    in this part you will be taught you all required hardware and software assets of the computer and how they work inside of the computer in detail before moving to the programming. Then you will have a full of knowledge about the computer.

  • Part-2

    This part will cover all the Java basics like data types, variables, if else, looping, method calling, classes, interfaces etc in detail.

  • Part-3

    In this part you will learn All the OOP related things like OOP concepts, and exception handling etc. With the ending of this second part, you will have a better understanding of basic programming and how it works.

  • Part-4

    After collecting that knowledge and practicing, we move to the next part of the course, and then you will learn file management and database management database connection management to protect the clients' data in the real world.

  • Part-5

    In the final part, we learn programming design-patterns and one real-world application will be implemented by the instructor by using design-patterns and all the knowledge that we have learned through the course. You will gain a lot of knowledge and experience from it, like how an application is built by a programmer in the real world.

Course Duration: This course will take approximately 6-8 weeks to complete, depending on your pace of learning.

Prerequisites:

Assessment and Certification: At the end of the course, you have to build a real-world application by yourself. It will be your final project to show your performance. And after completing the final project, you have to attend an interview that will be taken place by the programmer.lk. To obtain your certificate and the t-shirt, you have to pass the interview, and you should attend the interview within 3 months after ending the course.
